Mive API documentation:
 

Url : http://www.mive.in/

1. Authtoken
   end point: */authtoken/
   POST
   media type :application/json
   body:
        {
              "user": "admin",
              "password":"givememyfuckingtoken" 
              }
2. Products of a category
   end point */api/product/category/{categoryId}/
   GET
3. Get items of Cart
    end point */api/cart/cartitems/{cartId}/
    GET
4. Add multiple products to cart
    end point */api/addtocart/
    POST
     body :
         {
          "cartId":"{cartId}",
          "userId":"{userId}",
          "items":
                  {
                  "1":{
                       "qty":"{quantity}",
                       "productId":"{productId}"

                       }
                   "2": ......similar
                   "3"   .....similar
                   #add as many products as you want

                  }

         }
5. Update Cart
   end point */api/updatecart/ 
   POST
    body :
         {
          "cartId":"{cartId}",
          "userId":"{userId}",
          "items":
                  {
                  "1":{
                       "qty":"{quantity}",
                       "itemId":"{itemId}"

                       }
                   "2": ......similar
                   "3"   .....similar
                   #add as many products as you want

                  }

         }
6. Make order 
   end point */api/makeorder/
   POST
   body:
           {
           "userId":"{userId}",
           "cartId" :"{cartId}"

           }
7. Orders of a user
   GET
   end point */api/user/orders/{orderId}/
8. Items of a order
   GET
   end point api/order/items/{orderId}
9. Custom category products
   GET
   end point */api/user/customcategory/(?P<userid>.+)/$
10. Product Search API
   GET
   endpoint */api/search/?searchtext={searchtext}
11. Product Search for Title API
   endpoint */api/search/title/{searchtext}
11. Product Search for Description API
   endpoint */api/search/description/{searchtext}
